About The Project

Geo-Search is a tool that allows people to search for places
and extract information such as timezone, map and local weather conditions
like temperature, wind speed, and humidity.

User can search for a place of interest. User can view a map of a place with a map marker
identifying the location of the place of interest. User can see the local weather conditions
(wind speed, temperature, humidity and the current weather condition). User can convert the temperature between Celsius
and Fahrenheit metric system. User can share a map of a place of interest
or its local weather conditions to Facebook.
